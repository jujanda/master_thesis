\begin{abstract}
Diese Arbeit beschäftigt sich mit der vorgeschlagenen Formatvorlage für wissenschaftliche Arbeiten der Forschungsgruppe Systemsicherheit. Es steht Ihnen selbstverständlich frei, die Vorlage an Ihre Bedürfnisse (und Ihren Geschmack) anzupassen. Zudem enthält dieser Text einige spezielle Tipps und Richtlinien für DiplomandInnen. Falls Sie Fehler entdecken, Wünsche oder Anregungen bzgl. der Formatvorlage haben, so senden Sie diese bitte per eMail an \href{mailto:peter@syssec.at}{peter@syssec.at}).

\vspace*{2cm}

  \begin{itemize}
    \item Diese Vorlage kann ohne Änderungen mittels PDFLaTeX von MikTeX compiliert werden. Daher müssen auch die
        Abbildungen als PDF vorliegen.
    \item Muss zur Compilierung aber LaTeX genutzt werden, da psfrag (oder Ähnliches verwendet wird),dann darf das Package

    \item Damit das Abkürzungsverzeichnis erzeugt wird, muss ``nomenclature.bat'' aufgerufen werden.
    \item Einige Dokumente die den Umgang mit LaTeX und diversen Packages beschreiben, finden Sie im Verzeichnis ``README''.
